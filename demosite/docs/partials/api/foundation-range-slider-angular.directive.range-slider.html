range-slider
directive in module foundation-range-slider-angular

Description

Creates a Foundation range slider.

Usage

as element:
<range-slider
       [ng-model="{string}"]
       [vertical="{boolean}"]
       [min="{number}"]
       [max="{number}"]
       [step="{number}"]
       [precision="{number}"]>
</range-slider>

Parameters

ParamTypeDetails
ngModel
(optional)
string

Assignable angular expression to data-bind to.

vertical
(optional)
boolean

If present, creates a vertical slider instead of a horizontal one.

min
(optional)
number

Angular expression evaluating to the minimum slider value. Only evaluated once, when the slider is created.

(default: 0)

max
(optional)
number

Angular expression evaluating to the maximum slider value. Only evaluated once, when the slider is created.

(default: 100)

step
(optional)
number

Angular expression evaluating to the slider step value. Only evaluated once, when the slider is created.

(default: 1)

precision
(optional)
number

Angular expression evaluating to the precision, i.e. the number of decimal points to use. Only evaluated once, when the slider is created.

(default: 2)
